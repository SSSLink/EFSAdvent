SWTH (switch)
One person switch

Variable 1a:
The local variable number that is triggered or set by this actor.

Variable 2a:
Horizontal trigger area length

Variable 2b:
Vertical trigger area length

Variable 4:
0 = Default
1 = Remains active
2 = Two click sounds
3 = Two click sounds, Remains active
4 = Invisible trigger
7 = Invisible trigger area, use V2 as range.
